# 交易检查清单模板

## 盘前检查
- [ ] 看主要指数状态
- [ ] 看当天宏观事件日历
- [ ] 看财报日历
- [ ] 看强弱板块轮动
- [ ] 做观察名单
- [ ] 定义今天的不做条件

## 开仓前检查
- [ ] 有明确逻辑
- [ ] 有明确触发条件
- [ ] 有明确止损条件
- [ ] 有明确止盈思路
- [ ] 仓位大小合理
- [ ] 已检查事件风险

## 持仓中检查
- [ ] 当前操作符合原计划
- [ ] 没有情绪化加仓
- [ ] 没有报复性交易
- [ ] 总风险仍在控制范围内

## 盘后检查
- [ ] 记录全部交易
- [ ] 复盘错误与偏差
- [ ] 提炼经验
- [ ] 准备明日计划

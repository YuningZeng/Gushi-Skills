# A股中长线价值投资 Skills

一个用于沉淀 **A股中长线价值投资方法、公司研究框架、估值检查流程、组合配置规则、复盘模板与风险控制原则** 的仓库。

## 快速开始

### 如果你今天要研究一家公司
按下面顺序使用：

1. `skills/a-share-analysis.md`：先看 A 股市场与风格环境
2. `skills/company-research.md`：建立公司整体认知
3. `skills/fundamental-review.md`：检查财务与基本面质量
4. `skills/valuation-check.md`：判断估值与安全边际
5. `skills/investment-plan.md`：写投资计划
6. `templates/company-analysis-template.md`：沉淀成标准研究记录

### 如果你今天要检查一个持仓是否继续拿
按下面顺序使用：

1. `skills/holding-management.md`：判断持有、加仓、减仓还是退出
2. `skills/earnings-and-event-tracking.md`：检查财报与事件是否改变逻辑
3. `skills/valuation-check.md`：看估值是否已明显高估或低估
4. `skills/portfolio-risk-control.md`：看组合层面的集中度和风险暴露

### 如果你今天要复盘整个组合
按下面顺序使用：

1. `skills/a-share-analysis.md`：看宏观与市场环境
2. `skills/portfolio-allocation.md`：检查组合结构
3. `skills/periodic-review.md`：做周度 / 月度复盘
4. `templates/portfolio-review-template.md`：输出标准复盘记录

## 核心理念
- 买股票就是买企业的一部分
- 先看公司质量，再看价格是否合理
- 先评�� downside，再评估 upside
- 不把短期波动误判为长期价值变化
- 不为市场情绪支付过高价格

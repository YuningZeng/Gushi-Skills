# A股中长线价值投资 Skills

一个用于沉淀 **A股中长线价值投资方法、公司研究框架、估值检查流程、组合配置规则、复盘模板与风险控制原则** 的仓库。

## 仓库目标

把主观、零散、情绪化的投资决策，沉淀成：

- 可复用的 **投资技能（Skills）**
- 可执行的 **研究与决策流程**
- 可检查的 **估值与风控规则**
- 可复盘的 **持仓与组合模板**

## 仓库结构

- `skills/`：核心投资技能
- `templates/`：研究、复盘与组合模板
- `rules/`：纪律与原则
- `examples/`：示例输出

## 当前包含的技能

- 每日复盘
- 交易计划生成
- 风险控制检查
- 仓位管理
- 财报与宏观事件应对
- 市场分析
- 系统提示词
- A股市场分析
- 价值投资决策
- 基本面检查
- 公司研究
- 估值检查
- 组合配置

## 适用场景

- A股中长线选股
- 行业与公司研究
- 财报季跟踪与复盘
- 估值与安全边际检查
- 组合构建与再平衡
- 长周期持仓管理

## 核心理念

- 买股票就是买企业的一部分
- 先看公司质量，再看价格是否合理
- 先评估 downside，再评估 upside
- 不把短期波动误判为长期价值变化
- 不为市场情绪支付过高价格

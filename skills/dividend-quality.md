# 股息与分红质量检查 Skill

## 目标
判断一家公司分红和股息是否具有可持续性，以及分红政策是否真正体现股东回报，而不是一次性的表面友好。

## 适用场景
- 红利型公司筛选
- 高股息策略研究
- 国企 / 公用事业 / 金融等高分红行业分析
- ���期持仓前检查股东回报质量

## 输入信息
- 历年分红率
- 股息率
- 自由现金流
- 资本开支需求
- 负债水平
- 分红稳定性
- 是否有回购记录
- 管理层资本配置风格

## 检查流程
1. **看分红是否稳定**
   - 是持续分红，还是偶发高分红？
2. **看分红是否有现金流支撑**
   - 是真金白银分红，还是勉强分红？
3. **看分红是否影响长期竞争力**
   - 分红后是否压缩了必要投资？
4. **看管理层态度**
   - 是否真正重视股东回报？
5. **输出结论**
   - 分红质量高
   - 分红尚可
   - 分红不可持续

## 输出格式
- 分红稳定性
- 现金流支撑
- 资本配置评价
- 股东回报质量
- 综合结论

## 核心规则
- 高股息不等于高质量
- 分红必须建立在持续创造现金流的基础上
- 真正好的分红，是股东回报与长期发展兼顾

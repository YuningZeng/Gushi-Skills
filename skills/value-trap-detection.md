# 价值陷阱识别 Skill

## 目标
识别那些“看起来便宜、实际上可能越来越差”的公司，避免把低估值误判成价值机会。

## 适用场景
- 低 PE / 低 PB 公司筛选
- 周期股大跌后研究
- 市场情绪极差行业中选股
- 看似便宜但增长持续下滑的公司分析

## 输入信息
- 当前估值指标
- 历史营收和利润趋势
- 现金流情况
- 行业景气度
- 竞争格局变化
- 资本开支与负债情况
- 管理层资本配置记录
- 市场给低估值的原因

## 识别流程
1. **先问为什么便宜**
   - 是情绪错杀，还是基本面真的在变差？
2. **看盈利能力趋势**
   - 低估值是否对应长期回报率下降？
3. **看现金流和资产质量**
   - 是否存在利润表好看、现金流很差的问题？
4. **看行业与竞争**
   - 是周期底部，还是长期衰退开始？
5. **看管理层行为**
   - 是否乱投资、乱并购、回报股东意识薄弱？
6. **形成结论**
   - 可能错杀
   - 需要更多验证
   - 高概率价值陷阱

## 输出格式
- 低估值原因分析
- 基本面趋势
- 现金流与资产质量
- 行业与竞争风险
- 综合结论

## 核心规则
- 便宜是结果，不是理由
- 真正的机会来自“价格低于价值”，不是“价格低于过去”
- 价值陷阱最常见于逻辑恶化但估值看似诱人的公司

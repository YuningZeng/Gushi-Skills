# 投资计划 Skill

## 目标
在买入一家公司之前，把投资逻辑、估值判断、风险来源、资金安排和持有预期写清楚，避免因为市场波动而临时改变标准。

## 适用场景
- 首次建仓前
- 计划加仓前
- 准备把公司纳入核心持仓前
- 市场大跌后重新评估买入计划时

## 输入信息
- 公司名称 / 股票代码
- 所属行业
- 核心投资逻辑
- 当前估值水平
- 目标买入区间
- 关键风险
- 计划持有周期
- 计划仓位上限
- 触发继续买入的条件
- 触发放弃投资的条件

## 决策流程
1. **先写清为什么值得投**
   - 公司靠什么长期创造价值？
   - 行业空间和竞争优势是否支持长期持有？
2. **再看价格是否合理**
   - 当前价格是否给了足够安全边际？
3. **定义建仓节奏**
   - 一次建仓还是分批建仓？
   - 在什么情况下继续加仓？
4. **定义放弃条件**
   - 基本面恶化
   - 估值严重透支
   - 竞争格局逆转
   - 管理层治理问题暴露
5. **定义跟踪重点**
   - 未来几个季度重点关注哪些经营指标？

## 输出格式
- 投资逻辑
- 估值判断
- 买入区间
- 仓位计划
- 风险清单
- 放弃条件
- 跟踪重点

## 核心规则
- 先有逻辑，再有价格，再有仓位
- 买入计划要适应波动，而不是被波动打乱
- 不因为“跌了很多”自动等于“可以买了”

# A股市场分析 Skill

## 目标
建立适用于 A 股中长线投资的大环境分析框架，避免脱离制度、政策、产业周期与资金风格来孤立看公司。

## 适用场景
- 盘前或周度市场观察
- 行业配置前判断大环境
- 考虑新建仓或提高仓位前
- 市场大幅波动后的再评估

## 输入信息
- 上证指数 / 深证成指 / 创业板指表现
- 两市成交额变化
- 北向资金流向
- 风格偏好（大盘 / 小盘，成长 / 价值）
- 产业政策变化
- 利率与流动性环境
- 房地产、出口、消费、制造业景气信号

## 分析流程
1. **看指数，不只看涨跌**
   - 是普涨、结构性上涨，还是局部抱团？
2. **看成交与流动性**
   - 增量资金是否入场？市场是否具备持续性？
3. **看风格切换**
   - 当前更偏红利、周期、消费、科技，还是高股息防守？
4. **看政策与产业线索**
   - 是否存在政策强化的主线行业？
   - 是否有政策压制或监管扰动？
5. **看 A 股估值环境**
   - 当前市场风险偏好高还是低？
   - 优质公司是否整体被高估或低估？
6. **输出环境结论**
   - 适合积极布局
   - 适合精选个股
   - 适合保守等待

## 输出格式
- 市场环境概述
- 风格判断
- 主要政策与产业影响
- 适合关注的方向
- 仓位建议

## 核心规则
- A 股中长期投资离不开政策周期
- 不在风格极热时无脑抬高估值预期
- 先理解市场在奖赏什么，再决定买什么

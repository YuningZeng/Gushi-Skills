# 估值检查 Skill

## 目标
判断一家公司当前价格是否已经透支预期，是否存在足够安全边际，避免“买对公司却买贵了”。

## 适用场景
- 新建仓前
- 大涨后是否追价
- 大跌后是否加仓
- 持仓再平衡

## 输入信息
- 当前股价
- 总市值 / 市值区间
- PE / PB / PS / EV/EBITDA（视行业而定）
- 历史估值区间
- 未来 1~3 年增长预期
- 同行业可比公司估值
- 分红率 / 股息率
- 无风险利率环境

## 检查流程
1. **看静态估值**
   - 绝对估值高不高？
2. **看动态估值**
   - 当前价格隐含了多高增长预期？
3. **看历史分位**
   - 当前估值位于历史低位、中位还是高位？
4. **看横向比较**
   - 与同行比，是溢价还是折价？原因是什么？
5. **看安全边际**
   - 如果增长不及预期，估值有多大压缩空间？
6. **形成结论**
   - 低估可布局
   - 合理可跟踪
   - 偏贵需等待
   - 明显透支不参与

## 输出格式
- 绝对估值判断
- 相对估值判断
- 历史分位判断
- 安全边际评估
- 最终建议

## 核心规则
- 估值不是精确点，而是区间
- 高质量公司可以享受溢价，但溢价也有边界
- 价值投资不等于只买低 PE

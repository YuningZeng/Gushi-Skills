# 公司研究 Skill

## 目标
建立一套适合 A 股中长线投资的公司研究模板，帮助判断“这是不是一家具备长期投资价值的公司”。

## 适用场景
- 初次研究一家公司
- 建立公司档案
- 长期跟踪核心持仓
- 行业比较分析

## 输入信息
- 公司名称
- 股票代码
- 所属行业与细分赛道
- 主要产品 / 服务
- 核心客户
- 商业模式
- 竞争对手
- 管理层背景
- 历史财务数据
- 关键风险因素

## 研究流程
1. **公司是怎么赚钱的**
   - 收入来源是什么？
   - 利润来自哪里？
2. **行业空间够不够大**
   - 行业是成长、成熟，还是衰退？
   - 公司能否持续拿到份额？
3. **护城河在哪里**
   - 品牌、渠道、成本、技术、规模、牌照、客户粘性？
4. **管理层是否靠谱**
   - 战略是否清晰？
   - 资本配置能力如何？
   - 是否尊重中小股东？
5. **财务是否支持逻辑**
   - 收入、利润、现金流与回报率是否互相验证？
6. **风险是什么**
   - 政策风险
   - 周期风险
   - 竞争加剧
   - 技术替代
   - 治理问题

## 输出格式
- 公司概览
- 商业模式
- 行业与竞争格局
- 护城河分析
- 管理层评价
- 风险清单
- 是否进入观察池 / 核心池

## 核心规则
- 不能一句“赛道很好”代替公司研究
- 管理层质量会长期影响估值中枢
- 真正的护城河要能反映到财务结果里

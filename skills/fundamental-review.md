# 基本面检查 Skill

## 目标
用一套统一框架快速检查一家公司当前基本面是否健康、是否恶化、是否值得继续跟踪。

## 适用场景
- 看年报 / 中报 / 季报前后
- 准备新建仓前
- 股价大跌后确认逻辑是否变坏
- 持仓定期体检

## 输入信息
- 营收增速
- 归母净利润增速
- 扣非净利润增速
- 毛利率
- 净利率
- ROE / ROIC
- 经营现金流
- 资产负债率
- 应收账款 / 存货变化
- 分红与资本开支情况

## 检查流程
1. **收入端**
   - 增长是加速、放缓，还是转负？
2. **利润端**
   - 利润是否真实？是否明显依赖非经常性项目？
3. **盈利质量**
   - 毛利率和净利率趋势如何？
   - ROE / ROIC 是否稳定？
4. **现金流质量**
   - 利润是否能转成现金？
   - 经营现金流是否匹配利润？
5. **资产质量**
   - 应收、存货是否异常膨胀？
   - 负债结构是否安全？
6. **结论输出**
   - 基本面改善
   - 基本面稳定
   - 基本面走弱
   - 需要警惕财务质量问题

## 输出格式
- 收入质量
- 利润质量
- 现金流质量
- 资产质量
- 综合结论

## 核心规则
- 只看利润不看现金流，容易踩坑
- 基本面拐点往往先体现在质量指标，而不是股价
- 连续多个维度变差，通常比单一指标波动更危险

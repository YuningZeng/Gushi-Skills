# 组合配置 Skill

## 目标
帮助投资者建立适合 A 股中长线价值投资的组合结构，避免过度集中、风格漂移与单一逻辑失控。

## 适用场景
- 新建投资组合
- 季度 / 半年度组合复盘
- 持仓过多或过少时优化结构
- 大幅波动后再平衡

## 输入信息
- 总资金规模
- 目标收益与可承受回撤
- 持仓数量
- 各持仓权重
- 行业分布
- 风格分布（成长 / 价值 / 红利 / 周期）
- 单一公司最大权重
- 现金比例

## 配置流程
1. **先确定集中度**
   - 核心仓、观察仓、现金仓如何分配？
2. **看行业分散度**
   - 是否过度押注单一行业？
3. **看逻辑分散度**
   - 持仓背后的增长逻辑是否高度一致？
4. **看仓位上限**
   - 单一公司和单一行业上限是多少？
5. **看现金与再平衡空间**
   - 是否保留应对波动与错误修正的余地？
6. **输出组合建议**
   - 提高集中度
   - 降低集中度
   - 增配防守资产
   - 保持不变

## 输出格式
- 当前组合概况
- 行业与风格分布
- 集中度评估
- 风险点
- 调整建议

## 核心规则
- 组合不是股票清单，而是风险结构
- 过度分散会稀释认知优势
- 过度集中会放大判断错误

# 周度 / 月度复盘 Skill

## 目标
帮助投资者站在更长的周期上复盘投资行为，避免被日常波动牵着走，把注意力放回公司质量、估值与组合结构。

## 适用场景
- 每周复盘
- 每月复盘
- 财报季后集中复盘
- 组合明显跑输或跑赢后的归因分析

## 输入信息
- 当前组合收益
- 各持仓涨跌幅
- 组合权重变化
- 行业配置变化
- 新看到的公司信息
- 估值变化
- 是否有违反投资纪律的操作

## 复盘流程
1. **先看组合，不先看单只股票**
   - 当前组合的风险结构是否健康？
2. **看收益来源**
   - 收益来自基本面改善、估值修复，还是纯市场情绪？
3. **看错误来源**
   - 错在判断公司，还是错在买贵了，还是错在仓位配置？
4. **看是否偏离原则**
   - 是否因为短期波动而冲动操作？
5. **输出改进动作**
   - 下阶段应加强研究、优化仓位，还是提高现金比例？

## 输出格式
- 组合概况
- 收益归因
- 错误归因
- 原则偏离检查
- 下阶段改进点

## 核心规则
- 中长线投资更适合周度和月度复盘，而不是日内情绪复盘
- 关注“决策质量”，不要只关注“阶段盈亏”
- 复盘的目的是修正方法，不是自我批评表演

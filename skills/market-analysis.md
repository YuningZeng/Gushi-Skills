# A股价值投资市场分析 Skill

## 目标
从 A 股中长线价值投资视角，判断当前市场处于什么样的估值、风格与政策环境中，从而决定配置节奏和出手方向。

## 适用场景
- 月度或季度市场复盘
- 判断当前应进攻还是保守
- 调整行业配置前
- 大盘大幅波动后重估环境

## 输入信息
- 主要指数表现
- 市场成交额
- 北向资金趋势
- 红利 / 价值 / 成长风格表现
- 宏观经济指标
- 政策变化
- 利率与流动性环境

## 分析流程
1. **看宏观与流动性**
   - 宽松、平稳还是偏紧？
2. **看市场风格偏好**
   - 当前市场奖励的是确定性、分红、成长，还是主题情绪？
3. **看估值环境**
   - 市场整体是便宜、合理还是偏贵？
4. **看政策与产业主线**
   - 哪些行业获得长期支持？哪些行业面临约束？
5. **形成配置结论**
   - 更适合积极布局
   - 更适合精选个股
   - 更适合防守与等待

## 输出格式
- 宏观与流动性环境
- 风格偏好
- 估值环境
- 政策影响
- 配置建议

## 核心规则
- A 股中长线投资离不开政策、流动性与风格切换
- 市场便宜时重视勇气，市场狂热时重视克制
- 先决定环境适不适合出手，再决定买谁

# 市场分析 Skill

## 目标
建立一个从“大盘 → 板块 → 个股 → 执行”的统一分析框架，减少只盯个股、不看环境的错误。

## 适用场景
- 盘前市场判断
- 盘中主线跟踪
- 盘后风格复盘
- 判断该激进还是保守

## 输入信息
- 主要指数表现
- 成交量变化
- 市场广度
- 领涨 / 领跌板块
- 热点持续性
- 利率 / 宏观 / 政策背景
- 目标个股所处板块地位

## 分析流程
1. **看大盘环境**
   - 是风险偏好提升，还是风险偏好下降？
   - 指数上涨是权重带动还是普涨？
2. **看市场广度**
   - 上涨家数、下跌家数、创新高/新低情况如何？
3. **看主线板块**
   - 哪些板块得到资金持续认可？
   - 哪些只是日内脉冲？
4. **看目标个股定位**
   - 是板块龙头、跟风、补涨，还是边缘票？
5. **输出环境结论**
   - 适合进攻
   - 适合轻仓试错
   - 适合防守等待

## 输出格式
- 大盘状态
- 市场广度判断
- 主线板块
- 目标个股定位
- 操作环境建议

## 核心规则
- 先看环境，再做个股
- 个股强，不代表时机对
- 主线里找机会，杂毛里找风险

# 价值投资决策 Skill

## 目标
帮助投资者用“公司质量 + 估值 + 安全边际 + 持有逻辑”来做决策，而不是被短期价格波动牵着走。

## 适用场景
- 准备新建仓
- 持有过程中评估是否继续拿
- 股价大跌后评估是否加仓
- 组合里做优胜劣汰

## 输入信息
- 公司名称 / 股票代码
- 所属行业
- 核心业务
- 投资逻辑
- 竞争优势
- 财务指标
- 当前估值水平
- 预期增长
- 主要风险
- 计划持有周期

## 决策流程
1. **先判断是不是值得研究的公司**
   - 行业空间是否足够？
   - 商业模式是否清晰？
   - 赚钱逻辑是否可持续？
2. **判断公司质量**
   - 是否有护城河？
   - 管理层是否值得信任？
   - 现金流是否健康？
3. **判断价格是否合理**
   - 当前估值是低估、合理还是高估？
   - 市场是否已经提前透支预期？
4. **判断安全边际**
   - 如果未来低于预期，下行空间有多大？
5. **形成动作建议**
   - 建仓
   - 继续观察
   - 分批买入
   - 暂不参与
   - 持有
   - 减仓 / 退出

## 输出格式
- 投资逻辑
- 公司质量判断
- 估值判断
- 安全边际
- 关键风险
- 最终建议

## 核心规则
- 好公司不等于好价格
- 便宜不等于有价值
- 没有看懂的公司，不因为跌了就买
